# Matrix 数字雨 — 免费配套包

一片能跑、能重配色的 Matrix 风格代码雨，全部用脚本生成。不用插件，不用付费模板，无需安装任何东西。

## 包里有什么

| 文件 | 是什么 |
|------|--------|
| `code/demo_matrix_rain.jsx` | 自包含脚本，建出这片雨（基础版）：绿色等宽字符列坠落、底亮顶隐拖尾、轻微辉光、少数较亮领先列。 |
| `prompts.md` | 视频里真实用过的提示词，按顺序一步步把这片雨搭出来。 |

## 跑一下 demo

1. 打开 After Effects。
2. `文件 > 脚本 > 运行脚本文件...`，选 `code/demo_matrix_rain.jsx`。
   （如果你用 CLI 工作流：`./adobe_cli.sh ae exec-script ".../demo_matrix_rain.jsx"`。）
3. 出现一个叫 `Matrix_Rain_Demo` 的合成，已经在下雨了。按小键盘 `0` 做 RAM 预览。

## 调成你自己的

所有可调项都在脚本顶部的 `CONFIG` 块里：

- `fontSize`、`colSpacing` — 字符大小、墙有多密。
- `strandMin` / `strandMax` — 每条坠落字串多长。
- `speedMin` / `speedMax` — 坠落速度范围。
- `charset` — 下落的字符（试试纯数字：`"0123456789"`）。
- `palette` — 一个词重配色：在 `"green"` / `"amber"` / `"blue"` / `"multi"` 间切换（`presets` 里三种基色；`"multi"` = 多色混合，每列随机取一种色）。
- `glow.radius` — 泛光强度，压低一点保持锐利。

改一个值，再跑一次，整片立刻更新。这是一个自包含的脚本，不需要插件、不需要其他下载、不需要会员就能跑。

## 一个值得知道的点

拖尾用的是文字动画器的选择器，它真实的属性名是 `ADBE Text Range Shape`。很容易以为是 `Type2`（那个其实是 "Based On"），结果命令跑通了却什么都没发生。脚本用的是正确的名字，所以每一列都拿到了拖尾。
