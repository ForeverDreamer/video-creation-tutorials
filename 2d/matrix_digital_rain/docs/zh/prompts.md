# 提示词（按顺序）

这些是视频里真实用过的提示词，一步一步。每一句都在描述想要的**成品**，而不是点哪里。主角是 After Effects，脚本只是被写得更快。

> `code/demo_matrix_rain.jsx` 里的 demo 是基础版成品。下面这组提示词展示完整的搭建过程，包括亮度分层和一处重配色。

**1. 第一步 — 描述成品，不描述操作**

```
黑色画布，绿色等宽字符列，从上往下坠落
```

得到朴素的绿色字符列，已经在坠落。还没有辉光，没有拖尾。

**2. 加辉光**

```
加一个辉光，让字符发光绽放
```

第一次绽放过头，字符糊成一团。

**3. 把辉光压回去**

```
辉光太重了，半径降到 8，保持字符锐利
```

现在字符又锐利又有光感。

**4. 加上标志性拖尾**

```
加上标志性的底亮顶隐拖尾
```

这就是 Matrix 的样子：每条字串底部最亮，往上逐渐隐去。（提醒：文字动画器的属性名是 `ADBE Text Range Shape`，不是 `Type2`。）

**5. 加密度**

```
收紧列距、拉长字串，增加密度
```

字雨从稀疏变成一面密集的墙。

**6. 亮度分层**

```
加亮度分层：少数近白领先列，然后亮/中/暗三档绿
```

少数近白领先列，配上亮 / 中 / 暗三档绿。到这一步，一眼就是 Matrix。

**7. 一处控制全部**

```
把这些提进一个 CONFIG 块，让我在一处就能重配色
```

所有值都进到一个 CONFIG 块里。改一个词，整片重配色 —— 绿变琥珀变蓝 —— 或者把字符集换成纯数字。
